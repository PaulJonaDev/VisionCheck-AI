# VisionCheck-AI – Documentación Principal 
 
 Este documento es la fuente de verdad del proyecto VisionCheck-AI. 
 
 ## 1. Objetivo del Sistema 
 Tamizaje visual inteligente basado en análisis de patrones oculares. 
 
 ## 2. Componentes del Monorepo 
 - Frontend (Expo + RN) 
 - Backend (Node + Express) 
 - ML Pipeline (Python + TF) 
 - Infraestructura (Docker) 
 - Documentación centralizada 
 
 ## 3. Flujo General 
 1. Usuario captura imagen 
 2. Imagen se envía al backend 
 3. Backend valida calidad 
 4. Backend envía imagen a módulo ML 
 5. ML detecta patrones 
 6. Backend genera explicación 
 7. Frontend muestra resultados y guarda historial
